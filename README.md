# Java Full Stack Engineering (FSE) - Mandatory Hands-on Exercises

This repository contains all the mandatory and important hands-on exercises completed as part of the Java Full Stack Engineering learning path.

## Topics Covered

### 1. Design Principles & Design Patterns
- Singleton Pattern implementation
- Factory Method Pattern implementation

### 2. Data Structures & Algorithms
- E-commerce Platform Search Function
- Financial Forecasting

### 3. PL/SQL Programming
- Control Structures
- Stored Procedures

### 4. Test-Driven Development (TDD)
#### JUnit 5
- Setting up JUnit
- Assertions in JUnit
- Arrange-Act-Assert (AAA) Pattern
- Test Fixtures, Setup, and Teardown Methods

#### Mockito
- Mocking and Stubbing
- Verifying Interactions

### 5. Logging with SLF4J
- Logging Error Messages
- Working with Warning Levels

### 6. Spring Core & Maven
- Configuring a basic Spring application
- Dependency Injection
- Creating and configuring Maven projects
- Spring IoC Container configuration
- Constructor and Setter Injection
- Creating a Spring Boot application

### 7. Spring Data JPA with Hibernate
- Spring Data JPA Quick Example
- Difference between JPA, Hibernate, and Spring Data JPA
- Country Management Services
- Find Country by Country Code
- Add a New Country
- Query Methods in Spring Data JPA
- Object-Relational (O/R) Mapping
- HQL and Native Queries

### 8. Spring REST using Spring Boot 3
- Creating a Spring Web Project using Maven
- Loading Country data from Spring Configuration XML
- Hello World REST API
- Country REST API
- Fetch Country by Country Code
- JWT-based Authentication Service

### 9. Microservices with Spring Boot & Spring Cloud
- Creating Account and Loan Microservices
- API Gateway Implementation
- Eureka Discovery Server Setup and Service Registration

### 10. React
Completed multiple hands-on exercises covering:
- Components
- Props and State
- Event Handling
- Forms
- Routing
- Hooks
- API Integration
- Advanced React Concepts

### 11. Git
Hands-on exercises covering:
- Repository creation
- Branching and merging
- Commit management

- Collaboration workflows

### 12. Angular
Hands-on exercises covering:
- Components
- Data Binding
- Directives
- Services
- Routing
- Forms
- HTTP Client Integration

## Objective

The objective of this repository is to strengthen practical knowledge in Java Full Stack Development by implementing real-world exercises across backend, frontend, database, testing, microservices, and version control technologies.
